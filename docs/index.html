OpenSonATA Release 2.0

August 13, 2010

Notes

This is Release 2.0 SonATA, executables and the start of source code releasing. Please see the release roadmap for more details about what we are going to release and when.

This release contains the source code for these openSonATA applications:
  • The Waterfall Display
  • The Baseline Display
  • A new curses based information display called sonataInfoDisplay
Click Here for more details about the source code. Information about obtaining and installing the source code is included below.

For more information please visit setiQuest.org

Release 2.0 is intended to be an add on to Release 1.0. Release 1.0 was the binary release that supplied only the executables and source files to run the SonATA software in a limited scope on your computer. Release 2.0 is the start of SETI providing source code for the programs included in the binary release.
  • First, you need to install Release 1.0. Please follow the instructions at the setiCode Download page to download and install the Release 1.0. If you have not done so already, try to run the sonata-gen and sonata-vger scripts as explained in the Release 1.0 documentation. See the instructions at openSonATA Release 1.0 page.
  • To download the Release 2.0 software git clone git://github.com/setiQuest/setiCode.git
  • Copy the contents (overwriting) of the docs directory from the GIT repository into the docs directory of Release 1.0.
  • Copy the src directory from the GIT repository into the root directory of Release 1.0. (probably called OpenSonATA_1.0)
  • In your Release 1.0 directory cd src, then make.
  • Read the contents of the software page to get started and learn about the source code located in the src directory.

How You Can Help!

Please visit the source code page. The documentation for each software application source code contains a list of things that need work.
